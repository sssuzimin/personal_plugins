# 使用说明
# 1. 平时保持本插件开启，用于屏蔽开屏、底栏、内嵌广告及相关素材。
# 2. 需要“看广告获取下载资格”时：保持 NB 助手在前台，临时关闭本插件，等待数秒后再点“看广告”。
# 3. 广告播放完成并领取资格后，重新开启本插件。
# 4. 在规则行前加“#”表示停用；删除行首“#”表示启用。建议每次只改一条，便于排查误杀。
# 5. 本插件规则对设备上的所有 App 生效，DOMAIN-SUFFIX 规则尤其可能影响其他使用同一广告平台的 App。
# 6. 文末两条 /nb/app 规则是误杀留档，禁止启用。
